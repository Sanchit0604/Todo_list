TaskFlow is a fully interactive To-Do List web application built with vanilla HTML, CSS, and JavaScript (no frameworks or libraries). All six experiment tasks are implemented, including the bonus Task 6.

```
dynamic_todo_list/
├── index.html   ← HTML structure, semantic markup, Google Fonts link
├── style.css    ← All styling: layout, colours, animations, responsive
├── script.js    ← All interactivity: add, edit, delete, complete, stats
└── README.md    ← This file
```
---
How to Run
Open the `dynamic_todo_list` folder in VS Code
Install the Live Server extension (if not already installed)
Right-click `index.html` → "Open with Live Server"  
OR simply double-click `index.html` to open in any browser
No build step, no npm install, no dependencies.
